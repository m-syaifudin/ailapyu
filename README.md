# ailapyu - AI chat app
A lightweight app, fully containerized application featuring a localized AI service in VPS.<br>
The project pairs a Flutter Web frontend with a FastAPI backend, powered by a local Ollama LLM instance and a PostgreSQL database.

## Demo app
https://syaifudinprojects.my.id/

## Architecture Overview
User Browser -> Flutter Web -> FastAPI Backend -> Ollama LLM / PostgreSQL DB
